BASICS OF HTML

Webpage is constructed using HTML - Hyper Text Markup Language. Contents of webpage such as docuement, image, audio or video is accessed through HTML. Fundamental building blocks are as follow,

TAGS
Tags are used to format contents in HTML page. Format of specifying any tag is < NAME > Content < /NAME > (dont' consider white spaces, during code writing) where Name is Tag name, Content is any normal content or it could other tag as well. First Name is called opening tag and other one is closing tag.

ELEMENTS
It is composed of opening tag, content & closing tag. There are two types of it Inline & Block.

Lets discuss some of tags
1] To make any content bold, we use b tag. For Example UDACITY NANODEGREE
2] To make italics, we use em tag. For Example UDACITY NANODEGREE
3] To give line break, we use br tag. This tag is called void tag because there is no closing tag means no contents.
4] To insert link for other webpage, we use anchor tag. For Example UDACITY NANODEGREE

In summary, webpages are constructed using HTML language. TAGs & ELEMENTs are basic building blocks of particular HTML page.
